🖥️ Computer Lab Management System
📌 Description

Computer Lab Management System is a web-based application developed using HTML, CSS, Python (Django), and SQLite to efficiently manage and monitor computer lab operations within an educational institution. The system provides a centralized platform to handle lab resources, track system usage, and manage users in a structured and organized manner.

The application allows administrators to add, update, and delete computer records, monitor system availability, and manage user access. It helps in maintaining accurate records of lab usage, reducing manual effort, and improving operational efficiency. The backend is powered by Django, ensuring secure data handling and smooth interaction with the SQLite database, while the frontend provides a clean and user-friendly interface.

This project demonstrates core web development concepts such as CRUD operations, database management, and structured application design using the Django framework.
